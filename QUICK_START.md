# 🚀 快速启动指南

## 一键启动（推荐）

```bash
# 在项目根目录运行
./start_server.sh
```

或者手动启动：

```bash
uvicorn main_sqlite:app --reload --host 0.0.0.0 --port 8000
```

## 访问应用

1. **前端页面**: http://localhost:8000
2. **API文档**: http://localhost:8000/docs
3. **健康检查**: http://localhost:8000/api/health

## 测试流程

### 1️⃣ 填写宠物信息
- 选择宠物类型（猫/狗）
- 输入品种（必填）
- 输入年龄（必填）
- 选择健康状况（必填，至少1项）

### 2️⃣ 选择产品
- 浏览产品列表
- 或使用"系统推荐"
- 或"手动输入"产品
- 至少选择1款产品

### 3️⃣ 开始分析
- 点击"开始分析"
- 观察进度条和百分比
- 等待分析完成（约1-2分钟）

### 4️⃣ 查看结果
- 查看产品排名（匿名）
- 切换Tab查看不同排名
- 点击"显示真实产品名"
- 点击"导出分析结果图片"

## 🐛 如果遇到问题

1. **后端无法启动**
   ```bash
   # 检查端口占用
   lsof -i :8000
   
   # 安装依赖
   pip3 install -r requirements.txt
   ```

2. **前端无法访问后端**
   - 确认后端服务正在运行
   - 检查浏览器控制台错误
   - 确认 `static/app_fixed.js` 中的 `API_BASE` 为 `http://localhost:8000`

3. **导出图片失败**
   - 检查网络连接（需要加载html2canvas库）
   - 查看浏览器控制台错误
   - 尝试刷新页面
